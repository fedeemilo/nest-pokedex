# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar 
```
yarn install
```

3. Tener Nest CLI instalado
```
yarn add -g @nestjs/cli
```

4. Levantar la base de datos (tener instalado docker!)
```
docker-compose up -d
```

## Stack usado
* MongoDB
* Nest
